# 初中核心知识交互学习网站需求文档

## 1. 项目定位

本项目是面向初中生的交互式知识学习网站。第一阶段从初中物理开始，通过可操作的实验、动画讲解、典型例题、分层练习和即时反馈，帮助学生理解核心概念并掌握考试题型。

网站不是普通教材电子化页面，而是围绕“现象观察、变量探究、规律总结、例题建模、练习反馈”的学习闭环设计。

## 2. 学科与首期范围

长期覆盖方向包括：物理、数学、化学、生物等初中核心学科。

首期聚焦初中物理，第一批核心知识点为：

- 浮力
- 欧姆定律
- 凸透镜成像

当前 MVP 先完整实现“浮力”知识点。

## 3. 单个核心知识点标准结构

每个核心知识点应包含以下模块：

1. 学习目标：说明学生学完后应能完成的任务。
2. 前置知识：列出需要提前掌握的概念。
3. 生活现象引入：用真实场景提出问题。
4. 定义与概念讲解：解释概念、公式、单位、适用条件。
5. 动画或交互实验：允许学生改变变量并观察结果。
6. 规律总结：用简短语言归纳核心关系。
7. 易错点与误区：说明常见混淆点和判断方法。
8. 典型例题考点：覆盖常见考试题型。
9. 解题思路模板：将题目拆成可复用步骤。
10. 实验探究：包含实验目的、器材、步骤、记录、结论和误差分析。
11. 分层练习：简单、中档、难题分层训练。
12. 即时反馈：学生作答后给出答案、解析和错误归因。
13. 小结与自测：帮助学生确认掌握程度。
14. 知识迁移：连接生活应用和综合题场景。

## 4. 浮力知识点 MVP

### 4.1 学习目标

学生完成本知识点后，应能够：

- 说出浮力的定义和方向。
- 理解浮力大小与液体密度、排开液体体积有关。
- 使用阿基米德原理计算浮力。
- 判断物体漂浮、悬浮、下沉的状态。
- 解释轮船、潜水艇、密度计等生活现象。
- 完成基础计算题、状态判断题和实验探究题。

### 4.2 核心公式

浮力大小等于物体排开液体所受的重力：

```text
F浮 = G排 = rho液 * g * V排
```

其中：

- `rho液`：液体密度，单位 `kg/m^3`
- `g`：重力常数，初中计算常取 `10 N/kg`
- `V排`：排开液体体积，单位 `m^3`

### 4.3 交互实验要求

页面需要提供一个“浮力实验台”，学生可以调整：

- 物体质量
- 物体体积
- 液体密度
- 浸入比例

系统实时展示：

- 排开液体体积
- 浮力大小
- 物体重力
- 当前状态：漂浮、悬浮或下沉
- 与参数联动的容器动画和物体位置

### 4.4 内容模块要求

浮力页面应包括：

- 生活问题：为什么钢铁做的船能浮在水面上。
- 定义讲解：浸在液体或气体中的物体受到向上的力。
- 动画讲解：浸入体积增大时，浮力随排水体积增大。
- 规律讲解：液体密度越大、排开体积越大，浮力越大。
- 易错点：浮力不是由物体质量直接决定；漂浮时浮力等于重力；浸没后继续加深，浮力不再因为深度增加而增大。
- 典型例题：直接公式计算、漂浮状态判断、实验数据分析。
- 解题模板：找状态、找体积、统一单位、套公式、比较浮力和重力。
- 分层练习：简单、中档、难题各至少一类。

### 4.5 验收标准

浮力 MVP 完成后应满足：

- 页面可直接在浏览器打开并运行。
- 交互参数改变后，浮力、重力、排开体积、状态同步更新。
- 页面包含完整的知识点讲解、易错点、例题、解题思路和分层练习。
- 练习题支持作答、判分、解析和错误归因。
- 计算逻辑有自动化测试覆盖。
- 题目判分逻辑有自动化测试覆盖。

## 5. 后续扩展

后续知识点可以复用同一套结构和组件模式。优先扩展：欧姆定律、凸透镜成像、压强、二力平衡、摩擦力、杠杆、家庭电路故障判断。
