# Middle School Learning

Interactive learning website for middle-school core subjects. The first implemented module is junior physics, starting with buoyancy.

## Current MVP

- Buoyancy concept explanation
- Interactive buoyancy simulator
- Formula and state judgment
- Mistake notes and solving template
- Layered exercises with instant feedback
- Node test coverage for calculations and quiz evaluation

## Run

Open `index.html` directly in a browser, or serve the folder with any static server.

## Test

```bash
npm test
```
